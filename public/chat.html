Words

Text of a maximum of 190 characters can be written at once. By pressing enter or when the maximum word count is met the participant is reassigned a new position in the text, resulting in an unexpected collaborative writing excersise.

Participants are visible to each other by their computer cursor.

A session is only active when at least one participant is present in it but the outcome only makes sense when multiple participants interact.

No content or data is recorded or archived. Each session is a moment in time, an immediate temporary space. An ephemeral descentralited library.

The sessions will be saved only in the cache of the computer in use. If the cache is cleaned, the content of that session will no longer be accessible from that computer.

The only way to permanently save the results of the session is by selecting the text and copy pasting it.

The content between participants is being shared by using a temporary memory enabled by WebSockets. WebSocket is a computer communications protocol, providing full-duplex communication channels over a single connection.
